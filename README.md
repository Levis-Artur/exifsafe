# ExifSafe

Free browser-based photo metadata remover. No upload, no tracking, no account.

## Features

- Remove common EXIF/GPS metadata from JPG, PNG and WEBP
- Process photos locally in the browser
- Download cleaned image
- Lightweight metadata marker report
- Free forever

## Privacy model

- No server upload
- No photo storage
- No analytics
- No telemetry
- No usage event collection
- No tracking cookies
- No account required

## No analytics or telemetry

ExifSafe intentionally does not include analytics, telemetry, tracking scripts, advertising
pixels, cookies or usage event collection. We do not count visitors or track how people use the
tool.

## Limitations

- Browser-based cleaning removes common metadata by re-exporting from pixel data
- It is not a forensic metadata audit
- Some unusual/proprietary metadata may require advanced tools
